# NodeJS CPU Bound Parallelism
test running cpu bound tasks in parallel

## Run one CPU bound task
```bash
$ node fibonacci.js
4807526976
[ 29, 612429822 ]
```

## Run multiple CPU bound tasks in parallel
```bash
$ node app.js
4807526976
[ 31, 543523455 ]
4807526976
[ 31, 683599624 ]
4807526976
[ 31, 872173672 ]
4807526976
[ 31, 951852169 ]
```
